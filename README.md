# Calculadora-de-n-meros-inteiros
Este Projecto é uma calculadora de números inteiros básica.

Objectivo?

-Somar os números inteiros dados.

Como faz?

-Recebe a 1º string;

-Reconhece como númeração;

-Recebe a 2º string;

-Reconhece como númeração;

-Soma as 2 string dadas;

Linguagem usada?

-Python.

OBS: Esta calculadora foi efectuada de um exercício de prática nos meus estudos de python.
